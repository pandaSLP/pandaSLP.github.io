hexo Next主题文章post信息去除图标

发表于 2017-01-19 | 分类于 日记 | | 阅读次数

通过hexo搭建个人博客有一段时间了，用的是Next主题，比较喜欢这种简约、大气的风格，唯一有点不爽的是，在文章的post信息一栏有几个讨人厌的小图标，像下图这样：

看页面源码，原来是用的Font Awesome，<i class="fa fa-calendar-o"></i>，在Next主题的文件夹里扒拉了半天，终于找到了修改的地方，贴出来供大家参考。
只需要找到 themes\next\layout_macro\post.swig 文件，找到对应的span标签，注释掉即可，比如想去掉发表日期前面的小日历图标，就把下面这块代码删除或者注释掉就行了。

同理，其他地方的布局和内容也可以修改的个性化一些噢。
